Guide: Pomodoro / Sprint Calculator

What it does

Outputs a simple schedule table for focus sessions.

How to use it

Set durations and number of pomodoros, then generate.

Formula

Total time ≈ n × (work + short break) + occasional long breaks.

Example

4×(25+5) minutes + 15 long break ≈ 2h 35m total focus block planning.

FAQs

Does this run a timer?

This page plans numbers; use a timer app to execute.

Long break every N pomodoros?

Classic Pomodoro uses a long break after four short cycles—adjust counts to match.

Deep work blocks?

Try longer work intervals if 25 minutes feels short for your task.
