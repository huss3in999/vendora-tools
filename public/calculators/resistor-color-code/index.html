Guide: Resistor Color Code

What it does

Combines digit bands and multiplier into resistance.

How to use it

Select band colors or digit values as provided, then decode.

Formula

R = (10×d1 + d2) × multiplier.

Example

Bands 4–7–×100 → 4700 Ω.
