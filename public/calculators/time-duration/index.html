Time Duration

Minutes between two times (same day or next day).

Educational estimates only. Not medical, legal, tax, or financial advice.

Guide: Time Duration

What it does

Find elapsed time between time-of-day values.

How to use it

Enter start and end times (24h or AM/PM as supported).

Formula

Convert to minutes, subtract, add 24h if end < start.

Example

22:00 to 02:00 next day → 4 hours.

FAQs

Does it handle DST jumps?

Wall-clock duration can be ambiguous on DST days; use UTC for precision.

Lunch breaks?

Subtract unpaid break minutes manually if your form doesn’t include a break field.

Crossing multiple days?

Some tools assume same calendar day unless you enable overnight mode.
