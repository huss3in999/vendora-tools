Guide: Temperature Converter

What it does

Linear conversions between temperature scales.

How to use it

Enter a value and select from/to units.

Formula

°F = °C×9/5+32; K = °C+273.15.

Example

100 °C → 212 °F.

FAQs

Why not Rankine?

Add if needed: R = °F + 459.67.

Negative Kelvin?

Kelvin is non-negative physically; invalid inputs should error.

Body temperature?

Normal ranges differ by site (oral, ear); conversion is scale math only.
