Guide: ROI Calculator

What it does

Converts profit or value change into an ROI percentage relative to cost.

How to use it

Enter cost basis and gain (or ending value minus cost), then calculate.

Formula

ROI = (Gain − Cost) ÷ Cost × 100%.

Example

Invest $1,000, sell for $1,250 → ROI = 25%.

FAQs

Does ROI include time?

Basic ROI does not; annualized return divides by holding period.

What about fees?

Include transaction costs in cost basis for realistic ROI.

Can ROI be negative?

Yes—if you lose money versus cost, ROI is negative.
