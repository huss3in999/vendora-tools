Average Fixed Cost Calculator

Calculate fixed cost per unit produced: divide total fixed cost for a period by the number of units produced in that period.

Inputs

Average fixed cost = total fixed cost ÷ number of units.

Calculated after you click Calculate.

Results

This is a unit-level allocation of fixed cost across the units you produced.

Average fixed cost —

Example

If total fixed cost is $8,000 and you produce 400 units, average fixed cost is $8,000 ÷ 400 = $20 per unit.

How it works

Average fixed cost = total fixed cost ÷ number of units. If units are zero, the average fixed cost is not defined.

Business tip

Track fixed cost per unit across volumes—average fixed cost usually falls as you produce more units in the same period.
